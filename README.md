# SYSTEM_SCHEDULE // IT_ENVIRONMENT

A highly interactive, terminal-themed web application designed to track and manage weekly academic class schedules. Built specifically for BSIT coursework, this project transforms a standard timetable into an immersive, cyberpunk-inspired developer environment.

## 🚀 Overview

This single-page application (SPA) replaces traditional static schedules with a dynamic, gamified experience. It features real-time audio synthesis, fluid animations, and a customized motivational system to encourage consistent study habits throughout the semester. 

## ✨ Key Features

* **Terminal UI & Glitch Physics:** A custom CSS-styled interface mimicking a dark-mode IDE, complete with hover-triggered cyber-glitch effects.
* **Web Audio API Synthesis:** 100% mathematically generated sound waves (no external audio files used). Features a background server hum, high-pitch data stream blips, mechanical keystrokes, and a high-tech success chime.
* **Interactive Matrix Canvas:** A continuous digital rain animation running in the background via the HTML5 `<canvas>` element.
* **GSAP Animations:** Smooth tab-switching, bouncy entrance effects, and cinematic "hard flash" transitions powered by the GreenSock Animation Platform.
* **Gamified "Execution" System:** Marking a class as complete triggers a particle confetti explosion and opens a modal with a typewriter effect, delivering randomized IT-specific motivations.

## 🛠️ Tech Stack

* **HTML5:** Semantic structure and containerization.
* **CSS3:** Advanced styling, CSS variables, glassmorphism (`backdrop-filter`), and keyframe animations.
* **Vanilla JavaScript (ES6+):** DOM manipulation, event handling, and audio engine initialization.
* **Web Audio API:** Native browser audio synthesis.
* **GSAP (GreenSock):** High-performance animation library.
* **Canvas-Confetti:** Lightweight particle generation.
